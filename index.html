Днепропетровск (Украина) современный завод оснащен передовым высокотехнологичным оборудованием и многоуровневой системой контроля качества, охватывающей каждый этап производства.

Каталог продукции

MULTI ZERO

Порог с нулевой высотой - условие, которое мы предъявляем к нашим системам. С откидной запорной планкой MULTI ZERO доступная среда становится реальностью для всех поколений: для детских колясок, инвалидных кресел и роляторов. Препятствий в виде порогов больше нет.

ПОДРОБНЕЙ

MULTI POWER без доп. элементов облегчения

​MULTI POWER повышает несущую способность полностью скрытых петель. Всего с одним дополнительным элементом можно изготовить створку весом до 180 кг. избежать перегрузки на нижней петле, легко собрать и смонтировать.

ПОДРОБНЕЙ

MACO openLife

Современная система контроля доступа с возможностью удаленного управления в режиме реального времени позволяет следить за тем, чтобы в дом не проникали посторонние. Доступ в комнаты для гостей, бюро, переговорные, производства итд. можно ограничить временными правами для сменяющихся групп людей.
